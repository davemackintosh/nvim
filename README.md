# nvim

A very opinionated, zero-config neovim setup. 

## Goals

A familiar editor that enables:

* Coding with
	* Autocomplete
	* Smart suggestions
	* Github CoPilot
* Good support for
	* Golang
	* Svelt (todo)
	* React (todo)
	* CSS (todo)
	* HTML (todo)
* Autoformatting
	* Go
	* Svelt (todo)
	* React (todo)
	* CSS (todo)
	* HTML (todo)
